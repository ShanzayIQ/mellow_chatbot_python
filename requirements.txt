# Chatbot API (FastAPI Backend)

### Setup:
1. Install Python 3.9 (cuz i was working on this one)
2. Run: `pip install -r req.txt`
3. Start server: `uvicorn app:app --reload` 

### Chat Endpoint:
- URL: `http://127.0.0.1:8000/chat`
- Method: POST
- Body:
```json
{
  "message": "Hello"
}

Response:
json
{
  "response": "Hi there!"
}
IMPORTANT [you can call your endpoint from Flutter using http.post]
